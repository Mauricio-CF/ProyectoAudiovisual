<p align="right"> <img src="https://komarev.com/ghpvc/?username=mauro-cf&label=Profile%20views&color=0e75b6&style=flat" alt="mauro-cf" /> </p>

<h3 align="left">Descripción del proyecto:</h3>
<h4>El proyecto Contenido Audiovisual nos permite categorizar, organizar de una manera mas eficiente varios contenidos multimedia
obteniendo información especifica que ayude a identificar a cada tipo de contenido</h4>

<h3 align="left">Objetivos:</h3>
<h4>
  <ol>Comprensión de Herencia y Polimorfismo</ol>
  <ol>Comprensión de Asociación, agregación y composición</ol>
  <ol>Impulsar la creatividad</ol>
  <ol>Desarrollo de habilidades Técnicas</ol>
  <ol>Resolución de Problemas</ol>
</h4>

<h3 align="left">Propositos:</h3>
<h4>
  <ol>Obtimizar la gestión</ol>
  <ol>Mejorar la accesibilidad</ol>
  <ol>Preservar el contenido</ol>
  <ol>Mantener el contenido audiovisual organizado</ol>
  <ol>Permitir que varios usuario trabajen de forma eficiente y organizada</ol>
</h4>


<h3 align="left">Clases y funcionalidad nuevas:</h3>
<h4>
  <ol>Clases NoticiasDeTV - Presentador</ol>
  <ol>Clases Podcasts - Host</ol> 
  <ol>Creación de metodo para agregar en todas las clases</ol>
</h4>
